Goal: Automatically add songs to a playlist based on the description of a youtube video

Program Steps:

1 - Access youtube links description
2 - Log into spotify account
3 - Create Playlist on spotify
4 - For each song listed in the description, add that song into the playlist

Usage Steps:

1 - Get apiKeys from youtube and spotify (spotify needs your personal account client secret)
2 - Insert apikeys into the relevant spots within the apiKeys.py
3 - Run program, it may not connect on first attempt just run the program again and it should work.
4 - Check spotify to see if playlist exists.
5 - Enjoy new music :)
